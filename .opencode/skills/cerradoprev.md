# cerradoprev

## Description
Skill focada no CerradoPrev - Hub completo para RPPS (Regime Próprio de Previdência Social) integrado ao COMPREV, parte do ecossistema GovTech da Cerrado Tech. Possui página de simulação 100% online hospedada no GitHub Pages dentro do repositório cerradofinancas.

## Instructions
1. Forneça informações sobre o CerradoPrev como solução de gestão previdenciária para órgãos públicos brasileiros
2. Explique a integração com COMPREV (Complemento de Previdência)
3. Destaque os módulos e funcionalidades principais:
   - **Motor de Concessão e Folha**: Automação na concessão de aposentadorias e pensões, memória de cálculo, pareceres jurídicos, minuta de portaria, paridade, margem consignável
   - **Arrecadação e Integração COMPREV**: Alíquotas patronais e do servidor, emissão de guias, conciliação bancária, parcelamentos, inadimplência
   - **Perícia Médica com IA**: Agendamento inteligente, laudos com assinatura digital ICP-Brasil, sugestão de CID via IA
   - **Portal do Servidor e Compliance**: Autoatendimento 24/7, simulador de aposentadoria, chatbot normativo (IA), compliance Pró-Gestão/CRP/LGPD
4. Ajude com dúvidas sobre legislação previdenciária municipal/estadual (RPPS), EC 103/19
5. Suporte no desenvolvimento de features do CerradoPrev
